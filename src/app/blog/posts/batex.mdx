## Concept
BatEx was an existing <Hi>Blender</Hi> addon that I've been using for a while to batch export FBX files. 
However, it had some limitations and bugs that I wanted to address, so I decided to create my own version of the addon with improved functionality and a more user-friendly interface.


The last update to the original BatEx was in 2019, and it was built for older versions of Blender, so I wanted to update it to be compatible with the latest versions and add some new features that I found useful in my workflow.

## Planning

There were several key features I wanted to include in the addon:
- **Single Export Mode**: Allow the tool to export a single FBX file containing all selected objects, which is useful for exporting entire scenes or groups of assets.
- **Re-export**: Re-export the last exported file with the same settings, saving time when making iterative changes to assets.
- **Unreal Settings**: Automatically apply Unreal Engine export settings to ensure compatibility and correct axes orientation when exporting assets for use in Unreal Engine.
- **User Interface**: Update the UI to be more intuitive and user-friendly, with clear options and settings for batch exporting.
- **Quick Access**: Add the export options to a separate panel, so users can quickly access the batch export functionality without having to navigate through multiple menus.


## Challenges

I had to update the addon to be compatible with the latest versions of Blender, which involved refactoring some of the code and updating the functions to match the new Blender API.
I also had to ensure that the addon used Blender's new Extensions system for registering the addon, which required some changes to the structure of the code.

## Programming
The addon is written in <Hi>Python</Hi>, using Blender's Python API to interact with the software and perform the batch export functionality.
I implemented the new features and improvements based on my experience using the original BatEx addon and the needs I had in my workflow when exporting assets from Blender to Unreal Engine.
I also added error handling and validation to ensure that the addon would work smoothly and provide feedback to the user if there were any issues during the export process.
Since the FBX exporter can change in Blender updates, I implemented a system to check for the enums available in the current version of Blender and adjust the available options accordingly, 
ensuring that the addon remains functional across different versions of Blender.

## Outcome

The updated BatEx addon is now available on <Hi>GitHub</Hi>, and it includes the new features and improvements that I set out to implement.
It has been tested in Blender 5 and is compatible with the latest versions of Blender
